Projects

These are some of the projects I have made over the years.

  • haC64

    haC64 is a Commodore 64 Emulator designed in VHDL on an FPGA (Xilinx Basys3). This was a hackathon project that I worked with a partner on, it is still under development. Once we have the 6502 (The processor for the Commodore) finished, we plan on workin on the VIC (The Video Controller) and using the Basys3's VGA port for video output.

  • Pipelined MIPS Microprocessor

    A Pipelined MIPS microprocessor designed in VHDL for the Xilinx Basys3 Development board. The procesor was designed in stages over the semester, and then combined later, when timing analysis was performed to find the highest clock frequency at which the processor could function. This was a project for Digital Systems Design 2 at RIT.

  • RGB LED API

    A small arduino project that I created to handle curl requests to change the color of an RGB strip connected to the arduino. Can also be used for individual RGB LEDs. This is the precursor to my next big project that will handle request on a Flask server and send them to the Arduino. Planning to have support for fading requests, and Google Home integration.
